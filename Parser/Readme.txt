Program made by Michael DeGraw 11/18/15

This program was developed with PyCharm Community Edition 4.5.4 (Python 3.5) under OS X El Capitan 10.11 beta

To run the program:
1) Put main.py in a folder with a .txt file containing the input to the program
2) Open a terminal and navigate to that folder
3) Enter the following (assuming you named your file text.txt:

python main.py text.txt

4) Output should be displayed on the console
